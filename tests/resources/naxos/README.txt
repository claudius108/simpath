Naxos is an XSLT 1.0 processor written in XSLT 1.0.

To invoke Naxos, put a reference to the stylesheet to run in an xml-stylesheet
processing instruction in the XML document to process, then run
  java com.icl.saxon.StyleSheet xml-document.xml naxos.xsl

Some samples can be found in the "samples" directory.
